superstore-sales-dashboard
An interactive sales analytics dashboard that visualizes business performance from the SuperStore dataset — built to track revenue, profit, and regional trends at a glance.

Overview
This project analyzes 10,000+ sales transactions from the SuperStore dataset and presents key business metrics through an interactive, executive-style dashboard. It covers revenue trends, profit margins, regional performance, and category-wise breakdowns.

Tools & Technologies
- **Excel** — Data source & initial data exploration
- **HTML/CSS** — Dashboard structure and styling
- **JavaScript** — Data processing and interactivity
- **Chart.js** — Data visualization (bar, line, pie charts)

 Key Features
- KPI cards showing total revenue, profit, and order metrics
- Regional sales performance comparison
- Category & sub-category wise revenue breakdown
- Interactive charts with hover details
- Responsive, dark-themed executive dashboard UI
 Files
- `SuperStore_Data.xlsx` — Raw sales dataset
- `superstore_dashboard.html` — Interactive dashboard (open in browser to view)

 How to View
1. Download `superstore_dashboard.html`
2. Open it in any web browser
3. Dashboard loads with all charts and KPIs

 Key Insights
- Identified top-performing product categories by revenue
- Analyzed regional profit variations to flag underperforming areas
- Found correlation between discount levels and declining profit margins

Skills demonstrated: Data Analysis, Data Visualization, Dashboard Design, Business Intelligence
